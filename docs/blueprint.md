# **App Name**: مساعد صيانة الأجهزة الطبية

## Core Features:

- إدارة حسابات المستخدمين: تسجيل دخول آمن، تسجيل حسابات للمستشفيات والمهندسين، وإدارة صلاحيات الأدوار (مستشفى، مهندس، مدير) بناءً على جدول المستخدمين.
- لوحة تحكم المستشفى: لوحة تحكم للمستشفيات لتسجيل وإدارة ملفاتهم الشخصية، إضافة أجهزتهم الطبية وتفاصيلها، وتتبع حالة هذه الأجهزة.
- إدارة طلبات الصيانة: نظام يسمح للمستشفيات بإنشاء طلبات صيانة جديدة للأجهزة المعطلة، وتحديد مستوى الأهمية، ومتابعة حالة الطلبات (مفتوح، قيد التنفيذ، مكتمل).
- أداة وصف الطلب بالذكاء الاصطناعي: أداة مساعدة بالذكاء الاصطناعي لمساعدة المستشفيات في صياغة أوصاف طلبات الصيانة بشكل أكثر دقة وشمولية، لتوفير المعلومات الضرورية للمهندسين.
- لوحة تحكم المهندسين والعروض: تمكين المهندسين من إنشاء وإدارة ملفاتهم الشخصية (تخصص، سنوات الخبرة، الملفات المعتمدة)، وتصفح طلبات الصيانة المتاحة، وتقديم العروض للطلبات المناسبة.
- الموافقات والتقييمات والإشعارات: آلية للمستشفيات لمراجعة عروض المهندسين وقبولها، ونظام للإشعارات في الوقت الفعلي حول تحديثات الطلبات والعروض، مع إمكانية تقييم المهندسين بعد اكتمال الخدمة.

## Style Guidelines:

- نظام الألوان العام: فاتح، ليعكس الوضوح والاحترافية، ويناسب بيئة العمل.
- اللون الأساسي: أزرق عميق يبعث على الثقة والاحترافية، ويستخدم للعناصر التفاعلية والرؤوس البارزة (#2862B4).
- لون الخلفية: أزرق باهت ودافئ جداً لخلق شعور بالراحة والنظافة، مناسب للقراءة الطويلة (#E7EBF3).
- لون مميز: سماوي نقي وجذاب يستخدم للتركيز على الإجراءات الهامة أو التنبيهات (#4ACADB).
- خط الرؤوس والنصوص: 'Inter' (sans-serif) لاضفاء مظهر عصري، حيادي وواضح، مع إعطاء الأولوية للسهولة في القراءة لجميع أنواع المحتوى.
- استخدام أيقونات خطية نظيفة وبسيطة تعكس الوظائف المختلفة (مثل المعدات الطبية، أدوات الإصلاح، أيقونات المستخدم) لتعزيز الفهم البصري.
- تصميم واضح ومباشر للواجهة مع التركيز على الاستجابة لتناسب مختلف أحجام الشاشات (كمبيوتر مكتبي وهاتف ذكي)، وتخطيط نظيف لسهولة التنقل.
- تأثيرات حركية خفيفة وسلسة للتغذية الراجعة البصرية عند التفاعل مع العناصر، مثل تحميل البيانات أو تقديم نموذج.
