
# صيانة بلس (Medical Maintenance Plus)

النظام المتكامل لإدارة صيانة الأجهزة الطبية في المستشفيات وربطها بالمهندسين المعتمدين.

## 🚀 كيفية الوصول للتطبيق واستخدامه
التطبيق متاح الآن كمنصة ويب متكاملة (Web App) وتطبيق قابل للتثبيت (PWA).

### 1. الرابط العام
بمجرد نشر التطبيق عبر Firebase Studio، سيكون متاحاً عبر الرابط الذي يوفره لك محرك النشر (App Hosting URL).

### 2. التثبيت كـ تطبيق (Install App)
التطبيق يدعم تقنية PWA، يمكنك تثبيته كبرنامج مستقل على جهازك:
- **على Android/Chrome**: اضغط على الثلاث نقاط في الزاوية واختر "Install App" أو "تثبيت التطبيق".
- **على iPhone/Safari**: اضغط على زر "Share" ثم اختر "Add to Home Screen".

## 🛠 الميزات الرئيسية
- **إدارة الأصول**: تسجيل ومتابعة الأجهزة الطبية.
- **طلبات الصيانة**: نشر طلبات صيانة بذكاء اصطناعي مساعد.
- **نظام المزايدة**: تقديم عروض فنية وأسعار من المهندسين.
- **الإشعارات اللحظية**: تنبيهات فورية (Push Notifications) عند أي تحديث.
- **لوحة إدارة**: مراقبة كاملة للنظام من قبل المدير.

## 💻 التقنيات المستخدمة
- **Frontend**: Next.js 15, Tailwind CSS, ShadCN UI.
- **Backend**: Firebase (Auth, Firestore).
- **AI**: Google Genkit (Gemini 1.5/2.0).
